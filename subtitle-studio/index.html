Subtitle Studio v4.0

Python 3.12 OpenAI Whisper FFmpeg / OpenCV PyInstaller CustomTkinter

A professional desktop subtitle editor with AI-powered transcription, built entirely in Python. A complete subtitle production pipeline in a single desktop application—load a video, generate subtitles with AI, edit them with frame-level precision, and export. Zero external tools needed.

Core Infrastructure

AI Subtitle Generation

OpenAI Whisper integration to generate subtitles from any video. Runs entirely locally.

  • Select model size (tiny → large)
  • Word-level timestamps for tight alignment
  • Auto-splits long sentences into readable cues
  • Conditioned on previous text for context

NLE-Grade Video Preview

Synchronized playback combining OpenCV visuals and Pygame audio.

  • Premiere-style waveform for audio peaks
  • Triangular playhead with visual drag scrubbing
  • Frame-level stepping (1/10 frame increments)
  • Adjustable preview resolution caps

Invisible Quality Control

An invisible automation engine (core/auto.py) that acts as a silent guardrail.

  • Auto-fix on load: Reindexes and strips whitespace
  • Auto-fixes overlaps and adjacent cue bleed
  • Auto-splits cues >84 characters or 2 lines
  • Pre-export cleanup for negative durations

Professional Export Engine

FFmpeg-driven rendering pipeline handling media, sidecar, and text outputs.

  • 4 Modes: Full, Split, Merge, Delete sections
  • Burned subtitles with custom styling
  • Lossless Option: CRF 0 + Ultrafast + FLAC
  • Subtitle-only: SRT, VTT, TXT, DOCX

Quality Control Thresholds

The integrated QC engine runs invisibly alongside the Whisper generation and export pipelines. It assesses Cues against strict broadcast constraints.

Check Type Threshold Resolution Action
Overlap Any negative gap Auto-fix (Trims previous cue)
Empty Cue No text present Auto-fix (Removes)
Negative Duration End < Start Auto-fix (Corrects)
Line Too Long > 42 characters Auto-split
High CPS > 25 chars/sec Manual Review Flag
Too Many Lines > 2 lines Manual Review Flag
Small Gap < 0.04s (1 frame) Manual Review Flag
