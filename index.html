sys.id_01

Resolving Workflow Bottlenecks with Custom Tooling

The Problem

Our production team faced a persistent incompatibility bottleneck: Adobe Premiere exports subtitles as .srt, but our hosting platform strictly requires .vtt. This forced a tedious, manual conversion process for every project.

The Execution

I built a custom, standalone macOS application using an AI-assisted Python workflow. The tool features a frictionless UI where users simply load files and convert instantly, handling bi-directional conversion with zero learning curve.

Insight: Automation turns a data-mapping chore into a non-event.
  • 95% Reduction in turnaround time
  • Instant File compatibility
  • Frictionless Adoption across team
sys.id_02

Standardizing Delivery & Mitigating Dependencies

The Problem

Large video assets required compression for online transfer, but we lacked a standardized enterprise tool. Relying on a patchwork of external third-party apps introduced workflow inconsistencies and security gray areas.

The Execution

I engineered a custom, terminal-based compressor using Python and FFmpeg. The system takes any video input and automatically encodes it into a web-ready .mp4, ensuring output quality is enforced by the pipeline.

Insight: Secure, internal utilities eliminate the risks of legacy freeware.
  • Zero External tool dependency
  • 100% Standardized outputs
  • Secure Local-only processing
sys.id_03

AI-Powered Audio Enhancement Suite

The Problem

Lacking access to high-end automated AI cleanup, noisy audio required hours of manual repair in Adobe Audition. This created a significant lag in delivery for executive interviews and high-stakes content.

The Execution

Leveraging DeepFilterNet and Python, I built an enhancement tool that runs through the terminal. Crucially, I added a variable slider for granular control, allowing users to dial in the exact level of AI cleanup needed.

Insight: AI is most powerful when paired with human creative control.
  • 80% Faster audio cleanup
  • Granular Creative control
  • Enabled Same-day delivery
